# What are PETs?

## 1. Privacy-Preserving Computation (Вычисления с сохранением конфиденциальности)

Это общий термин, описывающий методы, позволяющие выполнять вычисления над данными без раскрытия самих данных. Включает в себя:

*   **Differential Privacy (Дифференциальная приватность)**
    Математически строгий подход, при котором в данные или результаты запросов добавляется шум, чтобы предотвратить идентификацию отдельных лиц. Широко используется в статистике, машинном обучении и аналитике (например, Apple и Google применяют её в своих системах сбора данных).
*   **Homomorphic Encryption (Гомоморфное шифрование)**
    Позволяет выполнять вычисления над зашифрованными данными без их расшифровки. Результат вычисления, после расшифровки, совпадает с результатом, полученным над открытыми данными.
*   **Secure Multi-Party Computation (SMPC)**
    Несколько сторон могут совместно вычислить функцию над своими входными данными, не раскрывая эти данные друг другу.
*   **Trusted Execution Environments (TEE)**
    Использование защищённых аппаратных сред (например, Intel SGX), где данные обрабатываются в изолированной «энклаве», недоступной даже операционной системе.

## 2. Data Anonymization / De-identification (Обезличивание данных)

Это более традиционный подход, при котором персональные данные модифицируются так, чтобы нельзя было идентифицировать субъекта:

*   Удаление прямых идентификаторов (имя, email, ИНН и т.д.)
*   Обобщение (например, возраст указывается как диапазон: 30–40 лет)
*   Подавление (удаление редких значений)
*   k-анонимность, l-разнообразие, t-близость -- формальные модели обезличивания

⚠️ Однако классическое обезличивание не всегда надёжно -- существуют атаки по повторной идентификации (re-identification attacks).

## 3. Federated Learning (Федеративное обучение)

Подход в машинном обучении, при котором модель обучается на устройствах пользователей, а не на централизованном сервере. Исходные данные не покидают устройство, отправляются только обновления модели. Часто комбинируется с дифференциальной приватностью.
